# PullChain_Copy

《Pull Chain》是作者 Steven Miller 倾力打造的免费类推箱解谜游戏。我对里面关卡的设计惊为天人，并由衷地佩服作者对游戏机制变态般的运用能力。我为了更好地了解熟稔游戏机制，用 Claude 帮我编写了这个地图编辑器，还原了游戏直到第6大关的游戏机制。十分推荐大家体验。

## 项目概述

这是一个**像素风格的地图编辑器**，用于复刻和探索 Pull Chain 的核心游戏机制。纯前端单页应用，打开 `map-editor.html` 即可使用，无需构建工具。

### 技术架构

纯前端单页应用，打开 `map-editor.html` 即可使用，无需构建工具或服务器。JS 按职责拆分为三层 12 个小文件，通过普通 `<script>` 标签按依赖顺序加载。

```
map-editor.html        → 入口页面，声明共享 DOM 引用，按序加载所有 JS
map-editor.css         → 像素风暗色主题样式

js/engine/             → 纯逻辑游戏引擎（零依赖）
  constants.js         →   常量定义
  Tile.js              →   地块数据结构
  entities.js          →   实体定义（主角/光球/木箱/雪块/飞蛾）
  state.js             →   全局游戏状态管理
  movement.js          →   移动规则（碰撞/高度差/链式推动）
  physics.js           →   绳子跟随、飞蛾飞行（含高度变化）、光球开关灯、升降墙逻辑、骑乘跟随

js/editor/             → 编辑器 UI 层（依赖 engine）
  editor-state.js      →   编辑器模式/状态管理
  actions.js           →   鼠标/键盘操作处理
  ui.js                →   工具栏按钮与状态栏

js/renderer/           → Canvas 2D 渲染层（依赖 engine + editor-state）
  tiles.js             →   地形地块渲染
  actors.js            →   实体角色渲染
  render.js            →   10 层渲染管线调度
```

### 已实现的游戏机制

**地形系统**
- **墙体** — 基础障碍物，砖块纹理渲染
- **斜角墙** — 四角缺口可选（TL/TR/BL/BR），角色可从缺口方向斜向穿过
- **高地** — 地面抬升一格，脚底高度 +1
- **天窗** — 取消该格的高度上限（天花板豁免），贴图为最外圈轮廓+中间透明

**特殊地块**
- **蜘蛛网** — 站在上面的实体无法移动
- **踏板** — 压力触发器，通过引线连接升降墙
- **升降墙** — 两种类型：上升型（默认下沉，触发后升起）和下降型（默认升起，触发后下沉）

**实体系统**
- **主角** — 玩家操控角色，方向键/WASD 移动
- **光球** — 通过绳子与主角相连，主角移动时跟随。可开关灯光，影响雪块和飞蛾
- **木箱** — 可推动的箱子，支持同格堆叠两层
- **雪块** — 光球点亮时，相邻雪块融化变成水渍
- **飞蛾** — 光球点亮时，沿同行/同列直线飞向光球，遇墙或实体阻挡则停。飞行中保持高度不变（不落地），只有升降墙格子或脚下实体会改变其高度；与球不同高度时不会飞行

**核心机制**
- **高度系统** — 每个实体有脚底高度（height），格子有地面高度（tileFootLevel）。放置时新实体自动站到已有实体顶部；实体可站在其他实体上方形成堆叠；移动时骑乘者跟随下层实体
- **高差阻挡** — 实体脚底低于目标地面时无法移动（如同墙壁）
- **链式推动** — 推动实体时连锁推动前方一整排实体
- **天花板机制** — 格子顶部高度上限为 2。升降墙激活后若顶部高度超过天花板则阻止激活，防止实体被碾压
- **绳子跟随** — 球通过绳子跟随主角，受距离约束，可绕过障碍推拉

**编辑器功能**
- 键盘快捷键切换工具：`1` 画墙 `2` 斜角墙 `3` 擦除 `4` 蜘蛛网 `5` 踏板 `6` 升降墙 `7` 高地 `8` 放角色 `9` 放球 `0` 放箱子 `-` 引线 `=` 天窗
- ▶ 操控模式 — 方向键/WASD 移动角色，光球跟随，触发融化、飞蛾飞行等机制
- 鼠标拖拽连续绘制/擦除
- 悬停高亮（绿色可放置，红色阻挡）
- 地图导出为 JSON 格式（含全部格子属性、实体位置、引线连接）

### 渲染管线

11 个渲染 pass，从底到顶依次为：
1. 空地底色
2. 水渍
3. 高地
4. 踏板
5. 墙体 / 蜘蛛网 / 升降墙
6. 全部实体按高度排序绘制（箱子 → 光球 → 主角）
7. 天窗（最外圈轮廓，中间透明 — 在实体之上）
8. 网格线
9. 悬停高亮
10. 绳子
11. 引线（最上层）
