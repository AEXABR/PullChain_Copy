# PullChain_Copy

《Pull Chain》是作者 Steven Miller 倾力打造的免费类推箱解谜游戏。我对里面关卡的设计惊为天人，并由衷地佩服作者对游戏机制变态般的运用能力。我为了更好地了解熟稔游戏机制，用 Claude 帮我编写了这个地图编辑器，还原了游戏直到第6大关的游戏机制。十分推荐大家体验。

游戏链接：https://steven-miller.itch.io/pull-chain
（itch 现在需要魔法，一些游戏加速器也可加速 itch）

## 项目概述

这是一个**像素风格的地图编辑器**，用于复刻和探索 Pull Chain 的核心游戏机制。纯前端单页应用，打开 `map-editor.html` 即可使用，无需构建工具。

### 技术架构

```
map-editor.html   → 入口页面，引入依赖并启动
js/engine.js      → 纯逻辑游戏引擎（Tile/Entity/移动/高度/升降墙/引线）
js/renderer.js    → Canvas 2D 像素渲染器（12 层渲染管线）
js/editor.js      → 编辑器 UI（工具栏、鼠标/键盘交互、地图导出）
map-editor.css    → 像素风暗色主题样式
```

### 已实现的游戏机制

**地形系统**
- **墙体** — 基础障碍物，砖块纹理渲染
- **斜角墙** — 四角缺口可选（TL/TR/BL/BR），角色可从缺口方向斜向穿过
- **洼地** — 地面下沉一格，脚底高度 -1

**特殊地块**
- **蜘蛛网** — 站在上面的实体无法移动
- **踏板** — 压力触发器，通过引线连接升降墙
- **升降墙** — 两种类型：上升型（默认下沉，触发后升起）和下降型（默认升起，触发后下沉）

**实体系统**
- **主角** — 玩家操控角色，方向键/WASD 移动
- **光球** — 通过绳子与主角相连，主角移动时跟随。可开关灯光，影响雪块和飞蛾
- **木箱** — 可推动的箱子，支持同格堆叠两层
- **雪块** — 光球点亮时，相邻雪块融化变成水渍
- **飞蛾** — 光球点亮时，沿同行/同列直线飞向光球，遇墙或实体阻挡则停

**核心机制**
- **高度系统** — 每个实体有脚底高度（height），格子有地面高度（tileFootLevel）。实体可站在其他实体上方形成堆叠
- **高差阻挡** — 实体脚底低于目标地面时无法移动（如同墙壁）
- **链式推动** — 推动实体时连锁推动前方一整排实体
- **引线系统** — 编辑器内连接踏板和升降墙，踏板上的实体数量决定墙的升降状态
- **绳子跟随** — 球通过绳子跟随主角，受距离约束，可绕过障碍推拉

**编辑器功能**
- 键盘快捷键切换工具（1-3 地形，4-7 特殊地块，8-0 实体，- 引线）
- 鼠标拖拽连续绘制/擦除
- 悬停高亮（绿色可放置，红色阻挡）
- 地图导出为 JSON 格式（含全部格子属性、实体位置、引线连接）

### 渲染管线

12 层分层渲染，从底到顶依次为：
1. 空地底色
2. 水渍
3. 洼地
4. 踏板
5. 墙体 / 蜘蛛网 / 升降墙
6. 箱子实体
7. 网格线
8. 悬停高亮
9. 绳子
10. 光球
11. 主角
12. 引线（最上层）
